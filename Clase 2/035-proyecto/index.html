Cómo dibujar rasgos faciales femeninos

El dibujo de el rostro humano es una de las cosas más difíciles que podemos abordar cuando creamos una ilustración. El objetivo de este vídeo es mostrarte, de la forma más sencilla y fácil posible, cómo crear, paso a paso, un dibujo de unos rasgos faciales femeninos. Para ello, vamos a empezar a analizar los volúmenes de trabajo, para, a continuación, empezar a detallar y a sombrear cada una de las partes individuales. El objetivo de este vídeo no es dibujar un rostro entero, sino únicamente los rasgos faciales principales ojos nariz y boca, creando una ilustración bastante sugerente con el papel blanco de fondo.

Cómo dibujar una perspectiva de un punto de fuga

Existen varias formas de dibujar perspectivas. Estás perspectivas son especialmente útiles en el mundo de la arquitectura para representar pasillos, calles largas, y otros elementos similares. Este vídeo te resultará muy interesante para saber cómo dibujar este tipo de entornos arquitectónicos, utilizando únicamente un lápiz y un folio

Cómo dibujar una cara femenina
